#### 题目描述

给定K个整数的序列{ N1, N2, …, NK }，其任意连续子序列可表示为{ Ni, Ni+1, …, Nj }，其中 1 <= i <= j <= K。最大连续子序列是所有连续子序中元素和最大的一个， 例如给定序列{ -2, 11, -4, 13, -5, -2 }，其最大连续子序列为{ 11, -4, 13 }，最大和为20。

三种思路解决最大连续子序列和问题，最佳时间复杂度为O(N)

#### 题目描述

给定K个整数的序列{ N1, N2, …, NK }，其任意连续子序列可表示为{ Ni, Ni+1, …, Nj }，其中 1 <= i <= j <= K。最大连续子序列是所有连续子序中元素和最大的一个， 例如给定序列{ -2, 11, -4, 13, -5, -2 }，其最大连续子序列为{ 11, -4, 13 }，最大和为20。

#### 解法

- 解法一

解法一的思想即为从第一个元素开始遍历，遍历一遍之后找到最大连续子序列和的值，然后遍历第二个元素，以此类推直到最后一个元素。其思想类似于简单选择排序，在每趟遍历中找到序列和的最大值。

时间复杂度为O(N2)

```c++
int maxsequence(int arr[],int len)
{
	int max = arr[0];
	for(int i=0; i<len; i++)
	{
		int sum=0;
		for(int j=i; j<len; j++)
		{
			sum += arr[j];
			if(sum > max)
				max = sum;
		}
	}
	return max;
}
```

- 解法二

解法二的思想是分治法，把一个复杂的问题分成两个相同或相似的子问题。

最大连续子序列的和有三种情况

1. 在序列的左半部分
2. 在序列的右半部分
3. 既包含左半部分又包含右半部分

通过序列中间数向左左半序列的最大值、中间数向右右半序列的最大值以及左右序列最大和加起来的值，三者取其大，得出最大连续子序列的值。

时间复杂度为O(NlogN)

```c++
int maxsequence2(int arr[], int low, int high)
{
    if(low > high)	return 0;
    if(low == high) return arr[low];
	int mid = (low + high)/2;    
    
    //求左半序列最大值
    int lmax=arr[mid], lsum=0;
    for(int i = mid; i >= low; i--)
    {
        lsum += arr[i];        
        if(lsum > lmax)
            lmax = lsum;
    }

    //求右半序列最大值
    int rmax = arr[mid+1], rsum = 0;
    for(int i = mid+1; i <= high; i++)
    {
        rsum += arr[i];
        if(rsum > rmax)
            rmax = rsum;
    }

    //返回三者较大值
    return max3(lmax+rmax, maxsequence2(arr,low,mid), maxsequence2(arr, mid+1, high));
}

//递归比较三个数返回最大值
int max3(int i, int j, int k)
{
    if(i >= j && i >= k)
        return i;
    return max3(j, k, i);
}
```

- 解法三

解法三的思想是动态规划，把原问题分解为相对简单的子问题的方式求解复杂问题，常常适用于有重叠子问题和最优子结构性质的问题。

对于后一个数来说，倘若它前面序列的和小于0，那么这个数加上前面的序列结果必定小于这个数原来的值。同理，对后面的序列而言，倘若它前面的序列和小于0，加上前面序列的结果也必定小于后面序列的值。为了得到最大连续子序列，若前面序列求和结果小于0，应该舍弃前面的序列，否则就使后面的序列和变小了。

从第一项开始求和，若序列和小于0就丢弃从下一项继续求和，遍历一次就可以得出最大连续子序列。

状态转移方程： sum[i]=max(sum[i-1]+a[i],a[i])

时间复杂度为O(N)

```c++
int maxsequence3(int arr[], int len)
{
    int max_sum, max_seq;
    max_sum = max_seq = arr[0];     //初始化为最大值 arr[0]
    for(int i = 1; i < len; i++)
    {
        if(max_seq <= 0)			//如果前面序列和小于0，则丢弃前面序列从下一项开始遍历 
            max_seq = arr[i];
        else						//如果前面序列和大于0，则前面序列 + 下一项 
            max_seq += arr[i];
            
        if(max_seq > max_sum)		 //更新最大连续子序列和
            max_sum = max_seq;
    }
    return max_sum;
}
```

int maxsequence3(int arr[], int len)
{
    int max_sum, max_seq;
    max_sum = max_seq = arr[0];     //初始化为最大值 arr[0]
    for(int i = 1; i < len; i++)
